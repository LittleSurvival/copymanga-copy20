CopyManga Copy20 & Tachiyomi

A modern fork of the stevenyomi/copymanga extension, updated to support the new domain copy20.com and compatible with recent Tachiyomi variants.

拜託打顆星星吧 ⭐ ~

🧩 說明

CopyManga Copy20 是 stevenyomi/copymanga 的延伸版，由於原作者已一年以上未更新，敝人基於其插件進行逆向與修補，使擴充套件能夠支援新域名 copy20.com，讓使用者能夠繼續從 CopyManga 平台閱讀漫畫。

該專案正在積極維護中，請考慮為儲存庫加星以表示您的支持！

🔧 用法

若要在 Tachiyomi 或其分支版本（Tachiyomi J2K、Suwayomi、Tachidesk 等）中安裝 CopyManga Copy20 擴充套件，請按照以下步驟操作：

  1. 使用倉庫更新地址。 在 Tachiyomi 中，打開 瀏覽 選項卡 > 擴充套件 > 點擊 三點菜單 並選擇 「新增擴充套件倉庫」。將以下 URL 貼到倉庫欄位中： 
    https://raw.githubusercontent.com/LittleSurvival/copymanga-copy20/repo/index.min.json
    這會告訴 Tachiyomi 直接從 GitHub 獲取 CopyManga Copy20 擴充套件。
  2. 安裝擴充套件。 新增倉庫後，在擴充套件列表中搜尋「CopyManga Copy20」並點擊「安裝」。擴充套件將會下載並啟用。
  3. 備用方案：下載 APK。 如果新增倉庫對您不起作用，您可以從專案的 發布頁面 手動下載 APK 並安裝到您的設備上。

📖 什麼是 Tachiyomi?

Tachiyomi 是一款適用於 Android 的免費開源漫畫閱讀器。它讓您可以從各種來源線上閱讀漫畫或離線閱讀已下載的章節。以下是它的一些顯著功能：

  • 從許多不同來源線上閱讀，以及離線閱讀已下載的內容。
  • 可配置的閱讀器，具有多種檢視器、閱讀方向和其他設定。
  • 與流行的追蹤服務整合，例如 MyAnimeList、AniList、Kitsu、Shikimori 和 Manga Updates。
  • 用於組織您的圖書館的類別以及自動淺色/深色主題。
  • 預定的圖書館更新以及建立本地或雲端備份的功能。

CopyManga Copy20 作為一個來源插入 Tachiyomi。安裝後，您可以直接在應用程式中瀏覽、搜尋和閱讀來自 CopyManga/HotManga 網站的漫畫。

❓ Q & A

README 也提供了一些常見問題和疑難排解提示：

  • [New] : 為何把熱辣源併進拷貝，不做獨立新源? 由於兩者後端和漫畫存儲模式近乎一模一樣，且搬遷會導致某些無端麻煩，不如直接用api域名併入。
  • 舊單行本下載檔案無法識別。 目前唯一解為解除收藏 -> "進階>清除資料庫" -> 刷新章節 -> "進階>重新索引下載位置"，但這也會導致閱讀紀錄被清除，不推薦。
  • 普通章節檔案無法識別可能是簡繁轉換導致無法匹配。
  • 關於只能看五頁，請嘗試更換User-Agent(包括插件設定以及Mihon本身自帶的UA設定)。
